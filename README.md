Retail Supply Chain & Sales Analysis Dashboard
📊 Project Overview

This project focuses on analyzing retail supply chain and sales performance using Power BI.
The dashboard provides insights into revenue, profit margin, inventory performance, and regional sales trends.
🎯 Objectives

Track Sales & Revenue Performance

Monitor Inventory Levels

Identify Regional Sales Trends

Analyze Profit Margins

Identified top-performing regions and products
