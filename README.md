Retail Supply Chain & Sales Analysis Dashboard
📊 Project Overview

This project focuses on analyzing retail supply chain and sales performance using Power BI.
The dashboard provides insights into revenue, profit margin, inventory performance, and regional sales trends.

Tools Used- Power BI, Power Query, Data Modelling
Developed interactive dashboard to total sales, total profit, profit margin, total orders and total customers. 
Used Power Query for data transformation and DAX for KPI calculations. 
Identified sales trends and regional performance insights to support decision-making. 
Implemented dynamic filters and slicers to enable real-time data exploration and improved stakeholder reporting efficiency
